{# Introduction #}

Network Design and Architecture

Network design and architecture involves planning, implementing, and managing computer networks that meet organizational requirements for performance, reliability, and security.

{# Network Types and Topologies #}

1. Network Types and Topologies

Network Types

  • LAN (Local Area Network): Networks within a limited area
  • WAN (Wide Area Network): Networks spanning large geographical areas
  • MAN (Metropolitan Area Network): Networks spanning a city
  • SAN (Storage Area Network): Dedicated networks for storage

Network Topologies

  • Bus: Single central cable
  • Star: Central hub/switch connecting all nodes
  • Ring: Nodes connected in a circular pattern
  • Mesh: Nodes with multiple interconnections
  • Hybrid: Combination of different topologies
{# Hierarchical Network Design #}

2. Hierarchical Network Design

Three-Layer Model

  • Core Layer: High-speed backbone
  • Distribution Layer: Routing, filtering, and WAN access
  • Access Layer: User and device connectivity
